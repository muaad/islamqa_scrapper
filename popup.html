This Chrome extension helps you collect links to questions and answers on IslamQA
How to use it:
  • Select a category: Go to https://islamqa.info/en/tree or any of the categories pages for the other languages and select a category.
  • Add Links: Click on the Add To List button next to each question
  • View Links: Click on the See Your Questions button towards the top
  • Copy Links: Your links will be displayed in a popup. Each link will comprise the text of the question and a link to it. Select all the text and copy it.
Now, go share it and spread the knowledge!
